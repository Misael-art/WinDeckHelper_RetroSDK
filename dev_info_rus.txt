[ MinGW-w64 ]
- Бесплатная среда разработки программного обеспечения с открытым исходным кодом для создания нативных приложений Microsoft Windows.

[ Clang ]
- Бесплатный компилятор с открытым исходным кодом для C, C++ и других языков.

[ CMake ]
- Кроссплатформенный генератор систем сборки с открытым исходным кодом.

[ vcpkg ]
- Бесплатный менеджер пакетов C++ для получения и управления библиотеками.

[ SDL2 ]
- Кроссплатформенная библиотека, предназначенная для обеспечения низкоуровневого доступа к аудио, клавиатуре, мыши, джойстику и графическому оборудованию.

[ SDL2-TTF ]
- Библиотека для SDL2, которая позволяет использовать TrueType шрифты в ваших приложениях SDL.

[ ZLIB ]
- Библиотека сжатия данных, используемая многими программами.

[ PNG ]
- Библиотека для работы с изображениями в формате PNG.
